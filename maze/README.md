# maze_ros

An virtual maze for robot simulation with Python, pygame and Sockets.

## Requirements

- python >= 3.9.7
- pip

## Install poetry

Poetry is used to handle dependencies, so need to be installed.

```sh
pip install poetry
```

## Install dependencies

Poetry will install all dependencies.

```sh
poetry install
```

## Run

Run project.

```sh
poetry run start
```
